# 📊 Course Quality Analysis using Data Analytics

## 📖 Project Overview

This project analyzes educational course quality using SQL, Python, PostgreSQL, and Power BI.

The objective is to evaluate course performance, student ratings, completion rates, readability, grammar quality, and overall quality through an end-to-end Data Analytics pipeline.

---

# 🎯 Objectives

- Analyze Course Quality
- Compare Department Performance
- Measure Student Ratings
- Analyze Completion Rates
- Build Interactive Dashboard
- Generate Actionable Insights

# 📂 Dataset

The cleaned dataset contains approximately **1947 records** with attributes including:

- Course_ID
- Course_Name
- Department
- Student_Rating
- Completion_Rate
- Feedback_Count
- Grammar_Score
- Readability_Score
- Overall_Quality
